# 📄 Personal Profile Page

This project is a simple personal profile webpage created using HTML5. It presents basic information about a student, including a short biography, academic interests, and career goals.

## 📌 Project Description

The Personal Profile Page is a static web page designed to introduce an individual in a clear and structured format. It demonstrates the use of basic HTML elements and semantic tags to organize content effectively.

The page includes:

- A header section with the student’s name and program  
- A short bio highlighting background and interests  
- A section explaining the motivation for studying web design  
- A list of career goals  
- A footer with copyright information
